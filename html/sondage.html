Différence logiciel libre et opensource

Les deux sont semblable. Il repose tous les deux sur des crictères, l'un est défini par l'open source intiative (opensource) et l'autre par la Free Software Foundation (libre). On peut cependant remarquer que la FSF a plus une vocation idéologique et moins permissive que l'opensource.

Type correspondant, à chaque logicie

  • Windows -> Logiciel propriétaire
  • Mozilla Firefox -> Logiciel libre
  • Discord -> Logiciel propriétaire
  • Avast -> Logiciel propriétaire
  • Python -> Logiciel libre

Définition "format de fichier" ou "format de données"

Un format de fichier ou format de données est la manières dont les données sont agencées dans un fichier. Cela permet au logiciel utilisant les formats de pouvoir récupérer les données grâce à leur spécifications. Les données sont écrit d'une manière unique et décrite publiquement ou non pour chaque format. Par exemple, en France les adresses commencent par les numéros et noms de rue et se finissent par la ville. Cette façon décrire la donnée sur les lettres et colis peut s'apparenter à un format de données.

Format de fichier libre ou propriétaire

Un format de fichier libre ou ouvert et un format qui respecte certains crières. Il doit avoir une documentation technique publique, être interopérable, ansi qu'être sans restriction d'accès et de mise en œuvre. Tous formats ne respectant pas une de ces conditions et un format propriétaire ou fermé.

Qu'est ce qui rend un format propriétaire ?

  • Aucune spécification n'est disponible
  • La spécification est publique mais son utilisation est limitée
  • L'accès à la spécification du format est payant
  • Le format est breveté

Classement les formats de fichiers suivants selon leur type

  • PNG -> Format de fichier libre
  • PDF -> Format de fichier libre
  • RAR -> Format de fichier propriétaire
  • EPUB -> Format de fichier libre
  • PPT -> Format de fichier propriétaire
