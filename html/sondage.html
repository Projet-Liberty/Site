Différence logiciel libre et opensource

Les deux sont semblables. Ils reposent tous les deux sur des critères, l'un est défini par l'open source intiative (opensource) et l'autre par la Free Software Foundation (libre). On peut cependant remarquer que la FSF a plus une vocation idéologique et moins permissive que l'opensource.

Type correspondant à chaque logiciel

  • Windows -> Logiciel propriétaire
  • Mozilla Firefox -> Logiciel libre
  • Discord -> Logiciel propriétaire
  • Avast -> Logiciel propriétaire
  • Python -> Logiciel libre

Définition "format de fichier" ou "format de données"

Un format de fichier ou format de données est la manière dont les données sont agencées dans un fichier. Cela permet au logiciel utilisant les formats de pouvoir récupérer les données grâce à leurs spécifications. Les données sont écrites d'une manière unique et décrites publiquement ou non pour chaque format. Par exemple, en France les adresses commencent par les numéros et noms de rue et se finissent par la ville. Cette façon d'écrire la donnée sur les lettres et colis peut s'apparenter à un format de données.

Format de fichier libre ou propriétaire

Un format de fichier libre ou ouvert est un format qui respecte certains critères. Il doit avoir une documentation technique publique, être interopérable, ainsi qu'être sans restriction d'accès et de mise en œuvre. Tous formats ne respectant pas une de ces conditions est un format propriétaire ou fermé.

Qu'est-ce qui rend un format propriétaire ?

  • Aucune spécification n'est disponible
  • La spécification est publique mais son utilisation est limitée
  • L'accès à la spécification du format est payant
  • Le format est breveté

Classement des formats de fichiers suivants selon leur type

  • PNG -> Format de fichier libre
  • PDF -> Format de fichier libre
  • RAR -> Format de fichier propriétaire
  • EPUB -> Format de fichier libre
  • PPT -> Format de fichier propriétaire
